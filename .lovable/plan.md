

# CodeQuest — Plataforma de Ensino de Programação Gamificada

## Visão Geral
Uma plataforma vibrante e gamificada para aprender programação, inspirada no Duolingo, com cores vivas, badges, barras de progresso, animações e elementos lúdicos que tornam o aprendizado divertido.

## Design & Estética
- **Paleta**: Fundo escuro suave com acentos em roxo, verde neon, amarelo e azul vibrante
- **Tipografia**: Headings bold e arredondados, corpo clean
- **Elementos gamificados**: XP points, streaks, badges, níveis, barras de progresso animadas
- **Mascote**: Emoji/ícone temático de código como identidade visual
- **Animações**: Hover effects, fade-ins, micro-interações nos cards e botões

## Páginas & Seções

### 1. Landing Page (Hero)
- Hero impactante com headline motivacional e animação de código sendo escrito
- Contador de alunos/cursos/exercícios com animação
- Seção "Como funciona" em 3 passos visuais
- Depoimentos de alunos com avatares e badges
- CTA vibrante para começar gratuitamente
- Footer com links e redes sociais

### 2. Catálogo de Cursos
- Grid de cards coloridos por linguagem (Python 🐍, JavaScript ⚡, React ⚛️, etc.)
- Filtros por nível (Iniciante/Intermediário/Avançado) e categoria
- Cada card mostra: ícone da linguagem, título, nível (badge colorido), duração, número de alunos, barra de progresso
- Tags de "Popular", "Novo", "Em alta" com badges animados

### 3. Editor de Código Interativo (Demo)
- Layout split: instrução à esquerda, editor à direita
- Syntax highlighting com tema escuro
- Botão "Executar" com feedback visual (✅ correto / ❌ tente novamente)
- Barra de progresso da lição atual
- Sistema de hints/dicas
- XP ganho ao completar exercício (animação de +10 XP)

### 4. Dashboard do Aluno
- Header com avatar, nome, nível atual e barra de XP
- Streak de dias consecutivos (estilo Duolingo)
- Grid de badges/conquistas (desbloqueadas e bloqueadas)
- Trilhas de aprendizado com progresso visual (mapa de nós conectados)
- Cursos em andamento com % de conclusão
- Ranking/leaderboard semanal

### 5. Navegação Global
- Navbar com logo, links para páginas, notificações (sino com badge) e avatar do usuário
- Menu mobile responsivo
- Transições suaves entre páginas

## Dados Mock
